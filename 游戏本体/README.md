# 夺宝躲怪

## 启动

1. 安装依赖：
   pip install -r requirements.txt

2. 运行游戏：
   python main.py

## 操作

- 移动：`WASD` 或 方向键（支持8方向：上、下、左、右、左上、左下、右上、右下）
- 开始：进入后先按一次 `Space`
- 闪现：`Space`（开始后作为闪现键，冷却80秒）
- 退出：`ESC`

## 速度规则

- 英雄移动速度：4格/秒
- 怪物移动速度：4格/秒（到加速时间后翻倍）
- 闪现距离：8格

## 怪物机制

- 怪物1出生：在玩家周围 `4-8` 格范围内随机出生（且必须可通行）
- 怪物2出生：到达配置时间后，同样在玩家周围 `4-8` 格范围内随机出生
- 怪物追逐：每一步使用 BFS 搜索到达英雄的最短路径，并按路径前进

## 提示与UI

- 右侧信息栏会显示离开视野的怪物大致方位（红色 `!` + 方向箭头）
- 右侧信息栏会显示最近宝箱方位（黄色箭头）
- 右侧信息栏包含宝箱坐标列表，并高亮最近宝箱
- 游戏结束界面会显示本局地图编号（例如“第3号地图”）

## 计分规则

- 英雄每移动1格：`+1.5` 分
- 吃到宝箱：`+100` 分

## 地图来源

地图文件放在：`../地图/tensorImg`

支持文本/JSON/NPY中的0-1矩阵：
- `1` 是障碍
- `0` 是可通行

程序会随机读取一张可用地图；
如果没有有效地图，会自动生成：`fallback_map.txt`

## 录像内容

每一局录像目录包含：
- `map.json`：该局地图（完整128x128）
- `record.jsonl`：每秒状态（英雄、怪物、宝箱、buff、得分等）
- `summary.json`：最终结果（最终得分、结束原因、地图编号等）

录像目录位于：`../录像文件/<结束时间>/`

## 清理旧录像脚本

脚本：`cleanup_replays.py`

示例：删除 8 小时以前的录像目录（会询问确认）

```bash
python cleanup_replays.py --older-than 8h
```

示例：先查看将删除哪些目录，不实际删除

```bash
python cleanup_replays.py --older-than 8h --dry-run
```

示例：不询问直接删除

```bash
python cleanup_replays.py --older-than 2d --force
```

支持时间写法：`90m`、`8h`、`1h30m`、`2d` 等。

## 回放脚本（全局视野）

脚本：`replay_player.py`

播放最新一局录像：

```bash
python replay_player.py
```

播放指定录像目录：

```bash
python replay_player.py --replay "../录像文件/20260418_210500"
```

常用按键：
- `Space`：播放/暂停
- `Left/Right`：上一秒/下一秒
- `Up/Down`：调快/调慢播放速度
- `Home/End`：跳到开头/结尾
- `R`：从头播放
- `ESC`：退出
